## À propos du projet

Aura Text est un excellent éditeur de texte/Code construit avec PyQt6 et Python.

## Premiers pas

### Prérequis
- Windows 10 x64 ou ultérieur, une distribution Linux avec noyau 6.x ou ultérieur, ou macOS Ventura ou ultérieur
- Python 3.9 ou ultérieur
- (Recommandé) `python -m venv venv` puis `venv\Scripts\activate`
- Installer les dépendances via `pip install -r requirements.txt`
