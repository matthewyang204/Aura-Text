## Über das Projekt

Aura Text ist ein hervorragender Text-/Code-Editor, der eine breite Palette wichtiger Werkzeuge bietet. Es basiert auf PyQt6 und Python und nutzt die Leistungsfähigkeit und Flexibilität dieser Technologien.

## Erste Schritte

Lassen Sie uns Aura Text auf Ihrem PC einrichten!

### Voraussetzungen
- Windows 10 x64 oder neuer, eine Linux-Distribution mit Kernel 6.x oder neuer oder macOS Ventura oder neuer
- Python 3.9 oder neuer
- Python-Installation ist mit pip bootstrapped
- (Empfohlen) Ein frisches venv mit `python -m venv venv` erstellen und mit `venv\Scripts\activate` aktivieren
- Den Inhalt von `requirements.txt` via `pip install -r requirements.txt` installieren
