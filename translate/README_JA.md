## プロジェクトについて

Aura TextはPyQt6とPythonで構築された優れたテキスト/コードエディタです。

## 始め方

### 前提条件
- Windows 10 x64以降、Linuxカーネル6.x以降、またはmacOS Ventura以降
- Python 3.9以降
- `python -m venv venv`でvenvを作成し、`venv\Scripts\activate`で有効にすることを推奨
- `pip install -r requirements.txt`で依存関係をインストール
