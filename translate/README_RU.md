## О проекте

Aura Text — отличный текстовый/кодовый редактор на PyQt6 и Python.

## Начало работы

### Требования
- Windows 10 x64 или новее, дистрибутив Linux с ядром 6.x или новее, либо macOS Ventura или новее
- Python 3.9 или новее
- (Рекомендуется) создать venv `python -m venv venv` и активировать `venv\Scripts\activate`
- Установить зависимости `pip install -r requirements.txt`
